### 내용

워드 임베딩 결과물을 이용하여 각 단어간 거리가 가까울수록 큰 가중치를 부여하는 가중치 행렬(weight matrix)을 산출하고 동시 발생 행렬(co-occurrence matrix)과 요소별 곱하여 단어 네트워크를 구축하였다.

사람의 인지적 관점에서 판단했을 때의 문서 범주를 비교하여 제안하는 방법론의 성능을 평가하였다.

### 결론

단어 벡터들을 사용하여 단어의 문맥적 의미까지 고려한 단어 네트워크를 생성하고 해당 문서들에 대한 중요 범주를 추출하는 방법론을 제안

#### 단어 연관 그래프

전체 문서에 Word2Vec 기법을 적용하여 등장 단어들의 임베딩 행렬(embedding matrix)을 생성하였다. 이후 임베딩 행렬에 식 (3)을 사용하여 단어 간 거리가 가까울수록 큰 가중치를 부여하는 N by N 크기의 가중치 행렬을 산출하였다. 식 (3)의 d(xi, xj)는 서로 인접한 노드 i와 노드 j 사이의 거리이다.
이를 바탕으로 가중치 행렬과 단어 동시 발생 행렬을 요소 별 곱해주어 최종적으로 단어간 연관성이 높을수록 작은 값을 가지는 단어 네트워크를 생성하였다.


### 용어

C-BOW 모델의 단어 임베딩 절차

- 먼저 모든 단어를 one-hot-coding으로 벡터화한다.
  - 문장에 있는 하나의 단어를 출력층으로 두고 해당 단어 w(t)의 앞 뒤로 k개의 단어를 입력층으로 둔다.
  - k는 윈도우 크기(window size)이며 사용자가 지정해야 하는 하이퍼 파라미터(hyperparameter)이다.
  - 표현하고자 하는 단어 벡터의 차원 수가 N, 전체 단어수는 T 라고 했을 때, 입력층의 단어 벡터는 가중치 행렬 W(T by N)과 곱하여 투영(projection)한다.
  - 이후 은닉층과 출력층 사이의 가중치 행렬 W′(T by N)와 곱하면 출력층의 단어가 나온다.
  - 파라미터 θ일 때 출력층의 단어 w와 입력층의 단어 c가 나타날 확률인식 (1)을 최대화하도록 학습한다.

단어의 개수가 너무 많아질 경우 효율적인 학습을 위해 hierarchical softmax, negative sampling과 같은 방법론을 적용한다(Mikolovet al., 2013).

Skip-gram은 입력층과 출력층만 다르고 모든 절차는 같다.
