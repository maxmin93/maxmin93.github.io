## 비 학습 단어의 워드 임베딩 처리를 위한 전처리 기법

- Pre-processing Techniques of word embedding for Out-Of-Vocabulary
- 저자: 용자윤 김철연
- 년도: 2020.11
- 출처 : DB Pia (국내)

### 요약

- 목적: 사전에 준비되지 않은 단어(Out-of-Vocabulary) 에 대한 처리 방법
- 비교: 기존 Unkown Token, Byte Pair Encoding, MIMICK Model 방법보다 낫다
- 제안: BERT 모델을 이용해 대체후보순위를 이용한다

### 의견

- 관련 없음

### 초록

본 논문에서는 테스트 시 OOV가 포함된 문장이 주어졌을 경우, OOV를 예측하는 방안에 대하여 제시한다. 모델 데이터셋에 존재하지 않는 단어를 단순히 Unknown Token 또는 무작위로 선택된 임의의 단어로 대체하는 방식은 결과의 정확도를 떨어뜨린다. 문맥정보를 이용하여 OOV를 대체할 수 있는 적합한 워드 임베딩을 찾아낸다면 상황에 맞는 적절한 단어 대체가 가능할 것이다. 이를 위해 BERT모델을 이용하여 OOV를 대체할 수 있는 단어들의 순위 목록을 얻는다. 그 후 Word2Vec을 이용하여 목록에 해당하는 단어의 임베딩 가중합을 통해 대체할 워드 임베딩을 얻는다. 이를 통해 단순 OOV처리 방식보다 유의미한 결과를 얻을 수 있다.

### 내용

- 문제 : You can still [MASK] breakfast when you are slimming.
  - eat 대신에 사용할 단어
- 후보 : MASK = { have, make, get }
  - eat 의 단어임베딩 가중치합 상위 3개
- 방법 : BERT의 단어예측알고리즘 사용 --> word2vec 함수


### 용어

- Out-of-Vocabulary(OOV) : 준비된 사전에 존재하지 않는 단어
