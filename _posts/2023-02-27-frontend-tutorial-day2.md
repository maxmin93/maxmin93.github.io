## 0. CSS(Cascading Style Sheets) 3

### 1) CSS 버전

- CSS1 : 1996년 12월
  + 텍스트 서식, 폰트 설정, 마진 설정 같은 핵심 기능을 정의
- CSS2 : 1998년 5월, HTML4 지원
  + 다양한 콘텐츠 배치 기능을 강화했음
  + 미디어별 레이아웃과 스타일 지정
- CSS3 : 2005년 12월, HTML5 지원
  + 이미지를 사용하지 않고도 그라데이터, 음영효과 등을 사용할 수 있음
  + 텍스트 효과, 블럭 효과 지시자 추가
  + Multiple column layout 으로 다수의 컬럼에 텍스트를 배치할 수 있음

- element 선택자, ex) `p`
- id 선택자, ex) `#my-id`
- class 선택자, ex) `.my-class`
- attribute 선택자, ex) `img[src]`, `details[open]`
- Pseudo-class 선택자, ex) `a:hover`
